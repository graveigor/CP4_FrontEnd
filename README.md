# Checkpoint 4 - Front-End 🚀

Este projeto é uma aplicação web desenvolvida para o Checkpoint 4 de Front-End, focada em criar uma interface moderna, intuitiva e totalmente responsiva utilizando o ecossistema Bootstrap.

## 🔗 Link do Projeto
Acesse o deploy oficial aqui: [CP-4 Front-End](https://cp-4-front-end-inky.vercel.app/)

---

## 🛠️ Tecnologias e Ferramentas

Para o desenvolvimento desta interface, foram utilizadas as seguintes tecnologias:

* **HTML5 & CSS3**: Estruturação semântica e personalizações de estilo.
* **Bootstrap 5**: Framework CSS para desenvolvimento ágil de componentes e sistema de grelha (grid) responsivo.
* **JavaScript (ES6+)**: Lógica para interatividade e manipulação do DOM.
* **Vercel**: Plataforma utilizada para o deploy e hospedagem da aplicação.

---

## 📖 Funcionalidades e Destaques

* **Design Responsivo**: Graças ao sistema de *breakpoints* do Bootstrap, a aplicação adapta-se perfeitamente a telemóveis, tablets e desktops.
* **Componentes UI**: Utilização de componentes nativos como Modals, Cards, Navbar e Buttons para uma experiência de utilizador consistente.
* **Navegação Fluida**: Estrutura organizada para facilitar a leitura e interação do utilizador.
* **Estilização Customizada**: Além das classes utilitárias do Bootstrap, foram aplicados estilos personalizados para garantir a identidade visual do projeto.
